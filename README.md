# GitHub User Activity CLI

A simple Python command-line tool to fetch and display the recent activity of a GitHub user in the terminal.

## Features

- Fetches the latest events of a GitHub user via the GitHub API
- Displays actions such as:
  - Starred repositories
  - Push events
  - Opened issues
- Handles invalid usernames gracefully

## Usage
python3 githubactivity.py <username>
python3 githubactivity.py EmilyaHovsepyan
